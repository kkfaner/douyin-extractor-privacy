1. Information Collection

1.1 Information We Collect

This extension only collects the following necessary information:

  • User Account Information: When you choose to log in to the Hengchuang AI Tools website, we store your user ID, username, and email address for authentication and permission management.
  • Permission Status Information: Your plugin purchase status, usage period, and other permission-related information for feature access control.

  • Temporary Operation Data: Cached pending operations to ensure work progress is not lost after page refresh.

2. Information Usage

The information we collect is only used for the following purposes:

  • Authentication: Verify your login status and usage permissions
  • Feature Implementation: Save your preference settings and provide personalized experience
  • Permission Management: Control feature access based on your purchase status
  • Status Notifications: Remind you when permissions are about to expire

3. Data Storage

3.1 Local Storage

All data is saved on your device using Chrome browser's local storage API (chrome.storage.local), including:

  • User login credentials (token)
  • User basic information
  • Permission status data
  • Theme preference settings
  • Temporary operation cache

3.2 Document Processing

Important: All PDF documents you upload and process are completely processed locally in the browser using JavaScript and WebAssembly technologies. Document content never leaves your device and is not uploaded to any server.

4. Permission Explanation

This extension requests the following browser permissions with their purposes explained:

4.1 activeTab (Active Tab)

Used to access the content of the currently active tab, only when you actively use features like "Capture Screen" or "Add Current PDF".

4.2 scripting (Script Injection)

Used to inject necessary scripts into web pages to implement PDF detection, page screenshots, cropping tools, and other features. Only executes when you actively trigger related functions.

4.3 contextMenus (Context Menus)

Used to add quick action options to the browser's context menu for convenient access to extension features.

4.4 storage (Local Storage)

Used to save preference settings, login status, and other data on your device. All data is only saved locally and not synced to the cloud.

4.5 tabs (Tab Management)

Used to create and manage workspace tabs, implement page screenshot functionality, and pass messages between tabs.

4.6 notifications (Notifications)

Used to send important notifications when permission status changes, such as reminders when permissions are about to expire.

4.7 Host Permission (http://tool.hcceo.com/*)

Used to communicate with the Hengchuang AI Tools website's authentication system for user login verification and permission checking. Only accesses this specific domain.

5. Third-Party Services

5.1 Authentication Service

This extension uses the authentication service provided by the Hengchuang AI Tools website (tool.hcceo.com) for user login and permission management. When you choose to log in, your login information is transmitted through the website's secure channel.

5.2 No Other Third-Party Services

Except for the authentication service, this extension does not use any other third-party services, analytics tools, or advertising networks.

6. Data Security

We take the following measures to protect your data security:

  • Local Processing: All document processing operations are completed locally without uploading to servers
  • Encrypted Transmission: Communication with authentication servers uses HTTPS encryption
  • Minimum Permissions: Only requests the minimum permissions necessary to implement features
  • No Data Collection: Does not collect any document content or browsing behavior data

7. User Rights

You have the following rights:

  • Access Right: You can view data stored by the extension at any time (via browser developer tools)
  • Deletion Right: You can clear all data stored by the extension at any time or uninstall the extension
  • Logout Right: You can log out at any time to clear account information
  • Choice Right: You can choose not to log in and use the basic features of the extension
