🎯1. 扩展功能概述

"高对比度模式"是一款专为改善网页视觉体验而设计的Chrome浏览器扩展，主要面向视力障碍用户、色盲用户以及需要长时间阅读的用户。

主要功能：

  • 12种预设视觉模式：增强对比度、灰度、反转颜色、黑底黄字、夜间护眼模式、红绿色盲优化、蓝黄色盲优化、高亮模式、柔和模式等
  • 自定义滤镜调整：提供8个可调参数（对比度、亮度、饱和度、灰度、反转、色相旋转、棕褐色、模糊）
  • 网站独立配置：为不同网站保存独立的视觉设置
  • 快捷键支持：通过键盘快捷键（Ctrl+Shift+H / Command+Shift+H）快速切换模式
  • 全局应用：可选择将设置应用到所有打开的标签页
  • 配置管理：支持导出和导入配置文件

🛡️2. 数据收集与使用

✓ 我们的承诺

本扩展不收集任何用户数据。我们不会访问、记录或传输您的浏览历史、个人信息、网页内容或任何其他数据。

我们不会收集：

  • 浏览历史记录
  • 个人身份信息（姓名、邮箱、地址等）
  • 网页内容或用户输入
  • cookies 或追踪数据
  • IP 地址或地理位置
  • 设备信息或浏览器指纹
  • 任何形式的统计或分析数据

本地存储的数据：

扩展仅在您的本地浏览器中存储以下配置数据，这些数据永远不会离开您的设备：

  • 当前选择的视觉模式
  • 自定义滤镜参数设置
  • 各网站的独立配置
  • 用户偏好设置（如是否记住网站设置、是否全局应用）

🔑3. 权限说明

为了实现其功能，本扩展需要以下权限。我们在此详细说明每项权限的用途：

权限名称 使用目的 数据处理
storage 保存用户的视觉模式设置和自定义参数 仅本地存储，不上传
commands 启用键盘快捷键功能 无数据收集
<all_urls>
(访问所有网站)		 向网页注入CSS样式以应用视觉效果 仅修改CSS样式，不读取内容

📌 关于"访问所有网站"权限的说明

本扩展需要访问所有网站的权限，原因是：

  1. 核心功能需求：扩展需要向网页注入内容脚本来修改CSS样式，这是实现视觉辅助效果的唯一方式
  2. 通用性要求：用户可能在任何网站上需要视觉辅助，无法预先限定特定网站
  3. 隐私保护：虽然有访问权限，但扩展仅修改页面的视觉样式，不读取、存储或传输任何网页内容

💾4. 数据存储

存储位置

所有用户设置通过 Chrome 的 chrome.storage.sync API 存储，这意味着：

  • 数据存储在您的本地浏览器中
  • 如果您启用了 Chrome 同步功能，设置会在您的设备间同步（由 Google 处理，非本扩展）
  • 数据不会发送到任何第三方服务器
  • 扩展开发者无法访问您的数据

存储内容详情

数据项 说明 示例
globalMode 当前全局视觉模式 "contrast", "grayscale", "normal"等
customFilters 自定义滤镜参数 {"contrast": 120, "brightness": 110, ...}
siteConfigs 各网站独立配置 {"example.com": {"mode": "invert", ...}}
rememberSite 是否记住网站设置 true / false
globalApply 是否全局应用到所有标签页 true / false
lastActiveMode 上次使用的模式 "nightMode", "custom"等

数据删除

您可以随时删除扩展存储的所有数据：

  • 方法1：卸载扩展（所有数据将自动删除）
  • 方法2：清除浏览器数据时选择"扩展的 Cookie 和网站数据"
  • 方法3：通过 Chrome 开发者工具手动清除（chrome://extensions -> 详细信息 -> 清除存储数据）

🌐5. 第三方服务

✓ 完全独立运行

本扩展不使用任何第三方服务。

  • 不使用分析服务（如 Google Analytics）
  • 不使用广告网络
  • 不连接外部服务器
  • 不使用第三方 SDK 或库（除了Chrome API）
  • 不加载远程代码或资源
  • 不使用 CDN 加载外部脚本

所有代码都包含在扩展包内，安装后完全离线运行。
