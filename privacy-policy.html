隐私政策

感谢您使用「抖音主页视频无水印提取器」Chrome扩展程序（以下简称"本扩展"）。我们非常重视您的隐私保护，本隐私政策将向您说明本扩展如何收集、使用和保护您的信息。

⚡ 核心承诺：本扩展不收集、不存储、不上传任何用户个人隐私信息。所有数据处理均在您的本地浏览器中完成。

1. 数据收集

本扩展仅在以下情况下读取数据：

  • 仅在用户主动操作时：当您点击扩展图标、悬浮按钮或右键菜单时，扩展才会读取抖音用户主页上的公开视频信息
  • 仅读取公开内容：视频链接、视频标题、视频封面图等公开显示的信息
  • 不收集个人信息：不收集您的抖音账号、密码、手机号、身份信息或任何私密数据
  • 不追踪浏览行为：不记录您的浏览历史或访问记录

2. 数据使用

  • 本地处理：所有视频信息的提取、展示和处理均在您的本地浏览器中完成
  • 第三方API调用：当您点击"去水印"按钮时，扩展会将视频链接发送到第三方公开的视频解析服务（api.7a.ink、api.douyin.wtf）以获取无水印下载地址
  • 不上传到开发者服务器：本扩展没有自己的服务器，不会将任何数据上传到开发者
  • 单一用途：数据仅用于提取和展示视频信息，不用于任何其他目的

3. 数据存储

  • 仅存储配置选项：本扩展使用Chrome的本地存储（localStorage）仅保存您的界面配置选项，例如是否显示首次使用提示
  • 不存储视频数据：不保存您提取的视频链接、标题或封面图
  • 您可随时清除：卸载扩展后，所有本地存储的配置数据将自动清除

4. 第三方服务

本扩展调用的第三方视频解析API为公开服务，仅用于解析无水印视频下载链接。这些服务的隐私政策由其服务提供方制定，我们不对第三方服务的数据处理行为负责。

  • api.7a.ink - 视频解析服务
  • api.douyin.wtf - 视频解析服务（备用）

5. 权限说明

本扩展申请的权限及用途：

  • storage：保存用户配置选项
  • contextMenus：在抖音页面上添加右键菜单快捷方式
  • activeTab：读取当前标签页的视频信息（仅在用户主动操作时）
  • clipboardWrite：实现"复制链接"功能
  • downloads：下载无水印视频到本地
  • douyin.com：在抖音网站上显示悬浮窗口并提取视频信息

Privacy Policy

Thank you for using the "Douyin Video Watermark-Free Extractor" Chrome extension (hereinafter referred to as "this extension"). We highly value your privacy protection. This privacy policy explains how this extension collects, uses, and protects your information.

⚡ Core Commitment: This extension does not collect, store, or upload any user personal privacy information. All data processing is completed in your local browser.

1. Data Collection

This extension only reads data in the following situations:

  • Only when users actively operate: When you click the extension icon, floating button, or context menu, the extension will read public video information on the Douyin user page
  • Only read public content: Video links, video titles, video thumbnails, and other publicly displayed information
  • No personal information collected: Does not collect your Douyin account, password, phone number, identity information, or any private data
  • No browsing behavior tracking: Does not record your browsing history or access records

2. Data Usage

  • Local processing: All video information extraction, display, and processing are completed in your local browser
  • Third-party API calls: When you click the "Remove Watermark" button, the extension sends the video link to third-party public video parsing services (api.7a.ink, api.douyin.wtf) to obtain watermark-free download addresses
  • Not uploaded to developer servers: This extension does not have its own server and will not upload any data to the developer
  • Single purpose: Data is only used to extract and display video information, not for any other purpose

3. Data Storage

  • Only store configuration options: This extension uses Chrome's local storage (localStorage) to save only your interface configuration options, such as whether to display first-time usage prompts

  • You can clear at any time: After uninstalling the extension, all locally stored configuration data will be automatically cleared

4. Third-Party Services

The third-party video parsing APIs called by this extension are public services, only used to parse watermark-free video download links. The privacy policies of these services are established by their service providers, and we are not responsible for the data processing behavior of third-party services.

  • api.7a.ink - Video parsing service

5. Permissions Explanation

Permissions requested by this extension and their purposes:
