Privacy Policy

Last Updated: January 2025

Core Commitment: We highly value your privacy. This extension does not collect, store, or transmit your personal data.

1. Information Collection

1.1 Information We Do NOT Collect

Canva Bulk Background Remover Pro does NOT collect any of the following information:

  • Personal identification information (name, address, phone number, etc.)
  • Browsing history
  • Content of images you process
  • Your Canva account information
  • Device information or location data

1.2 Locally Stored Data

To provide services, the extension locally stores the following data on your device:

  • User Preferences: Your download folder name, fast mode settings, etc.
  • License Information: Used to verify your usage rights (stored locally only)
  • Processing Statistics: Number of processed images and usage stats (for display only)

Important: This data is only stored on your local device and is never uploaded to any server.

2. Permissions Explanation

This extension requests the following permissions, used solely for core functionality:

2.1 storage

Purpose: Locally save user settings and configuration information on your device.

Data Scope: Download folder name, fast mode settings, notification preferences, license information, etc.

2.2 downloads

Purpose: Save processed images to your specified download folder.

Data Scope: Only access download functionality, does not read your download history.

2.3 tabs

Purpose: Interact with Canva web pages to execute automated processing workflows.

Data Scope: Only access tabs under the Canva.com domain, does not access other websites.

2.4 notifications

Purpose: Send you notifications about processing progress and completion status.

Data Scope: Only sends local notifications, no data transmission involved.

2.5 Host Permissions

  • *.canva.com: Access Canva website to perform background removal operations
  • tool.hcceo.com: Verify license status (only when you log in)
  • canva-license-server.onrender.com: Backup license verification server

3. Data Usage

3.1 Local Processing

All image processing operations are performed on the Canva website. This extension only serves as an automation tool:

  • Does not upload your images to any third-party servers
  • Does not analyze or store image content
  • Only assists you in batch processing images on Canva

3.2 License Verification

When you activate the Pro version, the extension will:

  • Send your email address to the license server to verify purchase
  • Receive and locally store license status information
  • Not share any other information about you

4. Data Sharing

We do not share your data with any third parties.

  • Do not sell user data
  • Do not share information with advertisers
  • Do not provide data to analytics services

5. Data Security

We take the following measures to protect your data security:

  • All locally stored data uses Chrome's secure storage API
  • License verification uses HTTPS encrypted transmission
  • No sensitive information is hardcoded in the code
  • Regular extension updates to fix security vulnerabilities

6. User Rights

You have the following rights:

  • Access Right: You can view locally stored data by the extension at any time
  • Deletion Right: You can delete all local data by uninstalling the extension
  • Control Right: You can disable features like notifications in settings

7. Third-Party Services

This extension interacts with the following third-party services:

  • Canva.com: Image processing platform (subject to Canva's privacy policy)
  • Hengchuang AI Tools: License management service (only for purchase verification)

8. Children's Privacy

This extension is not intended for children under 13, and we do not knowingly collect personal information from children.

9. Privacy Policy Updates

We may update this privacy policy from time to time. For significant changes, we will:

  • Display notifications in the extension
  • Update the "Last Updated" date on this page
  • Publish update notes through the extension store

10. Contact Us

If you have any questions or suggestions about this privacy policy, please contact us:

  • Email: hengchuang2025@126.com
  • Website: tool.hcceo.com

Transparency Commitment: We are committed to complete transparency. If you have any questions about how we handle data, please feel free to contact us.
