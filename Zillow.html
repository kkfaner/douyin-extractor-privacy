1. Overview

Zillow Data Exporter (hereinafter referred to as "this Extension") is committed to protecting user privacy. This privacy policy explains how we collect, use, and protect your information.

2. Information Collection

2.1 Information We Collect

This Extension collects the following information:

  • Login Credentials: When you log in to tool.hcceo.com, we store your login status and username
  • Extension Settings: Your personal preference settings (such as pagination delay time)
  • Permission Information: Your account permission status and expiration time

2.2 Information We Do NOT Collect

This Extension does NOT collect or transmit:

  • Any property data you extract from Zillow
  • Your browsing history
  • Your personal identification information (except username)
  • Your payment information
  • Any other personal sensitive data

3. Information Usage

3.1 Local Storage

All data is stored in your browser's local storage, including:

  • Login status
  • User settings
  • Permission information

This data is stored only on your device and is not uploaded to any remote server.

3.2 Network Requests

This Extension only sends network requests to the following domain:

  • tool.hcceo.com: For user authentication and permission verification
    • Sent content: Login credentials
    • Received content: User permission status

This Extension does NOT send any data you extract from Zillow to any server.

4. Data Security

We take the following measures to protect your data:

  • All communication with the authentication server is encrypted via HTTPS
  • Login information is stored in Chrome's secure storage
  • Sensitive data is not exposed to third parties

5. Third-Party Services

This Extension accesses the following third-party services:

5.1 Zillow.com

  • Purpose: Extract property data
  • Permission: Read page content
  • Data Processing: All data is processed locally and not transmitted to our servers

5.2 tool.hcceo.com

  • Purpose: User authentication and permission management
  • Data Transmission: Only transmits login credentials and permission verification requests
  • Privacy Protection: Does not collect or store your Zillow data

6. Data Export

This Extension allows you to export extracted data as Excel files:

  • Exported files are saved only on your local device
  • We cannot access any data you export
  • You have complete control over the use and sharing of exported data

7. User Rights

You have the following rights:

  • Access Right: View all data stored by the Extension
  • Deletion Right: Uninstall the Extension at any time to delete all local data
  • Control Right: Complete control over when to use Extension features

8. Children's Privacy

This Extension is not directed to children under 13, and we do not knowingly collect personal information from children.

9. Policy Changes

We may update this privacy policy from time to time. Significant changes will be notified through Extension update notes.

This Extension complies with:

  • Chrome Web Store Developer Policies
  • Applicable data protection laws and regulations
  • User privacy protection best practices
