隐私权政策

最后更新日期：2024年11月

重要提示：本扩展程序非常重视您的隐私保护。我们承诺不会收集、存储或分享您的任何个人数据。

1. 信息收集

本TikTok视频下载工具扩展程序不会收集以下任何信息：

  • 个人身份信息（姓名、邮箱、电话等）
  • 浏览历史记录
  • 下载历史记录
  • 设备信息
  • 位置信息
  • Cookie或其他追踪技术

2. 数据存储

本扩展仅在您的本地浏览器中存储以下必要信息：

  • 授权信息：用于验证扩展的合法使用权限
  • 用户配置：您的个人偏好设置（如下载路径等）

所有数据均存储在您的本地设备上，不会上传到任何服务器。

3. 权限使用说明

3.1 declarativeNetRequest权限

用于修改TikTok视频请求的响应头，解决跨域访问限制，确保视频能够正常下载。此权限仅作用于TikTok域名。

3.2 webRequest权限

用于监听网络请求完成事件，自动检测TikTok页面，提供下载功能。不会拦截、修改或记录您的任何请求数据。

3.3 tabs权限

用于识别当前标签页是否为TikTok页面，以便提供相应功能。不会访问其他网站的标签页信息。

3.4 storage权限

用于在本地存储授权信息和用户配置，所有数据仅保存在您的设备上。

3.5 downloads权限

用于触发浏览器下载功能，将视频保存到您的本地设备。

4. 第三方服务

本扩展仅与以下服务进行通信：

  • TikTok.com：获取视频资源，这是扩展的核心功能
  • 授权服务器（tool.hcceo.com）：仅用于验证扩展的使用权限，不会传输任何用户数据

5. 数据安全

我们采取以下措施保护您的数据安全：

  • 所有数据仅存储在本地，不上传到云端
  • 不使用任何追踪或分析工具
  • 不与任何第三方共享数据
  • 代码开源透明，可供审查

6. 用户权利

您拥有以下权利：

  • 随时卸载扩展，删除所有本地数据
  • 通过浏览器设置清除扩展存储的数据
  • 选择不使用扩展的任何功能

7. 儿童隐私

本扩展不针对13岁以下儿童。我们不会故意收集儿童的个人信息。

8. 政策更新

我们可能会不定期更新本隐私政策。更新后的政策将在扩展中公布，建议您定期查看。

9. 联系我们

如果您对本隐私政策有任何疑问或建议，请通过以下方式联系我们：

  • 通过Chrome网上应用店的扩展页面反馈
  • 发送邮件至开发者邮箱

Privacy Policy

Last Updated: November 2024

Important Notice: This extension takes your privacy very seriously. We commit to not collecting, storing, or sharing any of your personal data.

1. Information Collection

This TikTok Video Downloader extension does NOT collect any of the following information:

  • Personal identification information (name, email, phone, etc.)
  • Browsing history
  • Download history
  • Device information
  • Location data
  • Cookies or other tracking technologies

2. Data Storage

This extension only stores the following necessary information locally in your browser:

  • Authorization Information: Used to verify legitimate usage rights of the extension
  • User Preferences: Your personal settings (such as download path, etc.)

All data is stored on your local device and is never uploaded to any server.

3. Permission Usage

3.1 declarativeNetRequest Permission

Used to modify response headers for TikTok video requests to resolve cross-origin access restrictions and ensure videos can be downloaded properly. This permission only applies to TikTok domains.

3.2 webRequest Permission

Used to monitor network request completion events and automatically detect TikTok pages to provide download functionality. It does not intercept, modify, or record any of your request data.

3.3 tabs Permission

Used to identify whether the current tab is a TikTok page to provide appropriate functionality. It does not access tab information from other websites.

3.4 storage Permission

Used to store authorization information and user preferences locally. All data is only saved on your device.

3.5 downloads Permission

Used to trigger the browser's download functionality to save videos to your local device.

4. Third-Party Services

This extension only communicates with the following services:

  • TikTok.com: To retrieve video resources, which is the core functionality of the extension
  • Authorization Server (tool.hcceo.com): Only used to verify extension usage rights, no user data is transmitted

5. Data Security

We take the following measures to protect your data security:

  • All data is stored locally only, not uploaded to the cloud
  • No tracking or analytics tools are used
  • No data is shared with any third parties
  • Code is open and transparent for review

6. User Rights

You have the following rights:

  • Uninstall the extension at any time to delete all local data
  • Clear extension-stored data through browser settings
  • Choose not to use any features of the extension

7. Children's Privacy

This extension is not directed at children under 13 years of age. We do not knowingly collect personal information from children.

8. Policy Updates

We may update this privacy policy from time to time. Updated policies will be published in the extension, and we recommend that you review them regularly.

9. Contact Us

If you have any questions or suggestions about this privacy policy, please contact us through:

  • Feedback on the extension page in the Chrome Web Store
  • Email to the developer's email address
